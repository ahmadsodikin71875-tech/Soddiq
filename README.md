# Soddiq

Game mancing sederhana berbasis web.

## Cara menjalankan

1. Buka file `index.html` di browser.
2. Tekan tombol `Mulai Permainan`.
3. Gunakan tombol `Cast` untuk melempar pancing.
4. Tekan `Reel` untuk menarik pancing dan mencoba menangkap ikan.
5. Kumpulkan poin untuk meningkatkan bait dengan tombol `Upgrade Bait`.

## Fitur

- Memulai permainan dengan skor kosong.
- Melempar dan menarik pancing.
- Berbagai tipe ikan dengan peluang dan nilai berbeda.
- Riwayat tangkapan terbaru.
- Upgrade bait untuk peluang menangkap ikan lebih tinggi.
